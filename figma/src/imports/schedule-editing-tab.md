Проверь текущий дизайн вкладки «Редактирование расписания». Многое уже реализовано, но не всё и не всегда правильно.
Твоя задача:

Проанализировать всё, что уже есть.
Всё, что реализовано неправильно или неполно — полностью переделать.
Всё, чего не хватает — добавить.
Работать ТОЛЬКО с функционалом и логикой интерфейса (никаких изменений шрифтов, цветов, отступов, иконок и общего стиля).

Полный функционал, который должен быть:
1. Общая шапка вкладки (всегда видна):

Выбор недели (селект или табы недель). Рядом явно помечать, какая сейчас неделя: «Красная неделя» / «Синяя неделя» и отдельно выделять текущую неделю (например, текст «Текущая: Красная»).
Кнопка «Сбросить до выгруженного» (возвращает расписание к последнему выгруженному состоянию).
Кнопка «Автоматическая генерация» — при нажатии выпадающее меню с выбором:
• Для всего университета
• Для выбранного преподавателя
• Для выбранного института
Кнопка «Выгрузить».

2. Переключение между тремя таблицами:

Три большие кнопки-таба: «Преподаватели» / «Группы» / «Аудитории».
При переключении между табами все выборы сохраняются:
Выбранный преподаватель/группа/аудитория остаётся выбранным.
Выделенная пара (если есть) остаётся выделенной.


3. Фильтры перед выбором сущности (для каждого таба свои, необязательные):

Преподаватели: фильтры «Институт» и «Кафедра» → после них список преподавателей (поиск + выбор).
Группы: фильтры «Институт», «Степень образования», «Курс» → после них список групп.
Аудитории: фильтр «Корпус» (мульти) → после него список аудиторий.
Фильтры применяются мгновенно, список обновляется.

4. Добавление пары:

Кнопка «Добавить пару» (всегда доступна, можно также кликать по пустой ячейке таблицы).
При добавлении открывается боковая панель/модалка создания занятия.
В этой панели обязательно есть кнопка/вкладка «Просмотреть ограничения».

5. Ограничения (полностью переделать текущую реализацию):
При открытии «Ограничения» показывать ровно следующий список:

Преподаватель: доступность по времени
Аудитория: оборудование + доступность по времени
Группы: смена (1-я/2-я)
Нагрузка:
• Требование к оборудованию
• Параллельность (например, 2 пары английского у параллельных подгрупп должны идти одновременно)
• Двойная пара (пары, которые всегда должны ставиться вместе как блок)

6. Работа с существующей парой:

При клике по любой занятой ячейке в таблице пара выделяется.
Сразу появляются две кнопки: «Удалить пару» и «Переместить пару».
Выделение пары сохраняется при переключении между тремя табами (если перешёл на другой таб и вернулся — пара всё ещё выделена).
При нажатии «Переместить»:
В таблице автоматически подсвечиваются все возможные места:
• Красным — позиции, куда поставить нельзя (конфликты по преподавателю/группе/аудитории/ограничениям).
• Зелёным — идеальные позиции (без конфликтов, с учётом всех ограничений и предпочтений).
Клик по зелёной/допустимой ячейке перемещает пару.


7. Таблица расписания:

Классическая сетка (дни × пары).
Поддержка переключения между красной/синей неделей.
Все вышеописанные действия работают внутри таблицы.

Сделай вкладку максимально полной по инструментарию для сотрудника бюро расписаний. Проверь каждую деталь по списку выше, переделай неправильное и добавь отсутствующее.
