## 2. Тюнинг солвера под масштаб (опционально / исследовательское)

- [ ] На большом инстансе (~530 нагрузок) CP-SAT за 180 с возвращает `Unknown` (не успевает).
  Прунинг убрал краш по памяти, но полное решение такого размера за лимит времени —
  отдельная задача: подбор параметров поиска, эвристики первого решения, либо декомпозиция
  на меньшие подзадачи. На реальных данных через docker можно прогнать с бóльшим лимитом.
