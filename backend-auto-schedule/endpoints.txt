================================================================================
ЭНДПОИНТЫ ВЫБОРА: чьё расписание смотреть
================================================================================
(только справочники для каскадных фильтров в UI; сам вывод расписания — позже)

Правила:
  - Пути и query-параметры только латиницей (кириллица/пробелы в URL недопустимы).
  - Фильтрация — по ID (Guid), а не по названию. search= — по подстроке имени.
  - Все списки возвращают 200 OK + JSON-массив (пустой массив, если ничего нет).
  - Каскад повторяет доменную иерархию:
        Institute -> Degree (ступень) -> Course (курс) -> Group   (расписание группы)
        Institute -> Department (кафедра) -> Teacher               (расписание преподавателя)
        Building  -> Classroom (room)                              (занятость аудитории)


--------------------------------------------------------------------------------
ВЕТКА ГРУПП  (Институт -> Ступень -> Курс -> Группа)
--------------------------------------------------------------------------------

GET /api/institutes?search=
    Список институтов (факультетов).
    -> [ { id, name } ]

GET /api/degrees?instituteId=
    Ступени образования института (бакалавриат, магистратура, …).
    typeDegree — enum (Secondary|Bachelor|Specialist|Master|Postgraduate|Doctoral).
    -> [ { id, typeDegree, instituteId } ]

GET /api/courses?degreeId=&instituteId=
    Курсы (годы обучения) внутри ступени.
    -> [ { id, number, degreeId } ]

GET /api/groups?courseId=&degreeId=&instituteId=&search=
    Учебные группы — конечный выбор для расписания группы. Фильтры опциональны.
    shift — enum (First|Second|Evening).
    -> [ { id, name, shift, studentCount, courseId, courseNumber } ]


--------------------------------------------------------------------------------
ВЕТКА ПРЕПОДАВАТЕЛЕЙ  (Институт -> Кафедра -> Преподаватель)
--------------------------------------------------------------------------------

GET /api/departments?instituteId=&search=
    Кафедры. instituteId — фильтр по институту (опционально).
    -> [ { id, name, instituteId } ]

GET /api/teachers?instituteId=&departmentId=&search=
    Преподаватели — конечный выбор для расписания преподавателя.
    -> [ { id, name, departmentId, departmentName } ]


--------------------------------------------------------------------------------
ВЕТКА АУДИТОРИЙ  (Корпус -> Аудитория)
--------------------------------------------------------------------------------

GET /api/buildings?search=
    Учебные корпуса.
    -> [ { id, name } ]

GET /api/rooms?buildingId=&search=
    Аудитории — конечный выбор для занятости аудитории.
    -> [ { id, name, capacity, buildingId, buildingName } ]
